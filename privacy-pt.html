Política de Privacidade - Rutino Mio

Data da última atualização: 2026-02-04

1. Responsável pelo tratamento de dados

O responsável pelo tratamento de dados é Gregorio Visocchi Labs.
Contacto: GregorioVisocchiLabs@gmail.com

2. Âmbito e natureza da aplicação

Rutino Mio é uma aplicação para acompanhamento de rotinas diárias e hábitos. A aplicação não requer conta e não realiza registo de utilizadores.

3. Que dados processamos

3.1 Dados armazenados localmente (no dispositivo)

  • nomes de rotinas, horários e categorias,
  • estado de conclusão das rotinas,
  • preferências do utilizador (idioma, tema, configurações de notificações, consentimento de anúncios).

Estes dados são armazenados localmente na base de dados SQLite e em SharedPreferences e não são sincronizados com a nuvem.

3.2 Notificações locais

A aplicação programa notificações locais para lembrar as rotinas. As notificações são geridas exclusivamente no dispositivo e não requerem nenhum serviço externo.

3.3 Exportação e importação de dados

A aplicação permite exportar e importar dados de rotinas como ficheiro JSON. O ficheiro é armazenado localmente no dispositivo. Não temos acesso a ele.

3.4 Entrada de voz – microfone

A aplicação permite ditar os nomes das rotinas por voz. Ao ativar a entrada de voz, a aplicação solicita acesso ao microfone. O áudio é processado pelo sistema de reconhecimento de voz do sistema operativo do dispositivo (Google no Android, Apple no iOS) e pode ser enviado para os seus servidores para processamento. Não temos acesso ao áudio gravado nem aos resultados da transcrição.

3.5 Anúncios - Google AdMob

A aplicação utiliza o SDK do Google AdMob. No âmbito da exibição de anúncios, o Google pode processar, entre outros:

  • identificador de publicidade do dispositivo,
  • endereço IP,
  • dados do dispositivo e do sistema operativo,
  • informações sobre a interação com os anúncios.

Detalhes: https://policies.google.com/privacy

4. Finalidades e bases legais do tratamento (RGPD)

  • Prestação do serviço (art. 6.1.b do RGPD): dados locais necessários para o funcionamento das funcionalidades da aplicação.
  • Consentimento (art. 6.1.a do RGPD): no âmbito dos anúncios personalizados, se ativados conforme as configurações e os consentimentos do utilizador.
  • Interesse legítimo (art. 6.1.f do RGPD): exibição de anúncios não personalizados e manutenção do financiamento da aplicação.

5. Partilha de dados

Não vendemos nem partilhamos dados pessoais com terceiros. A exceção são os dados processados pelo Google no âmbito do Google AdMob.

6. Transferência de dados para fora do EEE

O Google pode processar dados fora do Espaço Económico Europeu. As transferências são realizadas conforme os mecanismos exigidos por lei; os detalhes estão disponíveis na política de privacidade do Google.

7. Período de conservação

  • Os dados locais são conservados até serem eliminados pelo utilizador ou a aplicação ser desinstalada.
  • Os dados publicitários são conservados conforme as políticas do Google.

8. Direitos dos utilizadores

O utilizador tem direito a:

  • aceder aos seus dados,
  • retificar os seus dados,
  • apagar os seus dados,
  • limitar o tratamento,
  • portabilidade dos dados (no âmbito dos dados locais),
  • oposição,
  • retirar o consentimento (sem afetar a licitude do tratamento anterior à retirada).

Tem também o direito de apresentar uma reclamação junto da Comissão Nacional de Proteção de Dados (CNPD).

9. Segurança

Os dados do utilizador são armazenados localmente no dispositivo. A aplicação não transmite dados do utilizador para os nossos servidores. Os dados podem ser transmitidos apenas para os serviços do Google no âmbito da publicidade (AdMob), se o utilizador os utilizar.

10. Crianças

A aplicação não é dirigida a crianças menores de 13 anos. Se é pai/responsável legal e acredita que uma criança forneceu dados, entre em contacto connosco.

11. Alterações à política

As alterações à política serão publicadas neste documento através da atualização da data.
